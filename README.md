## 🔍 Troubleshooting Common Issues

If the software fails to launch or work as expected, check these common fixes:

- Ensure your game is running in "Windowed" or "Borderless Windowed" mode. Most external trainers require these settings to display overlays correctly.
- Check that your display scaling in Windows settings is set to 100%. Incorrect scaling can prevent the overlays from appearing in the correct spot on your screen.
- Verify that no other game-related software is interfering with the trainer. Close any macro recorders or keyboard software while using the trainer.
- Update your graphic drivers from the manufacturer website if you experience screen flickering while the trainer is active.
